CSS란? Cascading Style Sheets

(Cascading = 폭포처럼 흐르는)

(Style Sheets = 여러 스타일 규칙을 포함)

폭포처럼 아래로 흐르는 스타일 규칙을 담고있는 시트

  마크업 언어가 실제로 화면에 표시되는 방법 (모양, 색상, 크기, 스타일..)을 기술하는 언어
  HTML코드가 어떤 스타일로 화면에 보이는가?


CSS 선택자가 핵심

    선택자란? 
    현재 HTML문서 내에서 특정 요소를 선택하는 문자(열)
    (요소 = 태그를 이용해 만든 모든 존재)

    스타일을 적용하고자 하는 요소를 선택하는 방법


CSS선택자의 종류

스타일 내에서 태그 이름 자체를 정의하던 것 tbody, thead가 어쩌구 했던 것

[작성법]

id선택자

HTML문서 내에서 id속성 값이 일치하는 요소를 선택하는 선택자
* id속성은 해당 element를 식별하는 역할을 한다
한 HTML 파일 내에서 같은 ID를 중복으로 가지게 하면 안된다.


작성법: id는 #아이디속성값 {css코드}와 같은 구조

  1. id선택자 test1
  2. id선택자 test2
  3. id선택자 test3
  4. id선택자 test4
