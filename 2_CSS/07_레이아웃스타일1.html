what is layout?

    layout(배치 or 정리)
    진짜 의미: 구성요소를 제한된 화면내에 효율적으로 배치하는 것

화면 배치방법 (형식) -> display속성

    
요소가 화면에 보여지는 형식 지정

   
   
 - block: 화면을 수직 분할 (행을 나누기)
    width또는 height로 설정 


    
   
 - inline: 화면을 수평 분할 
    width또는 height 못 건드림 


    
 - inline-block: inline의 수평분할 
      + 블록의 크기는 조절할 수도 있음 


    
 - none: 화면에 요소가 표시되지는 않으나
      존재하고 있는 상태 


    
 - flex: 요소의 정렬되는 방향이나 
      요소간의 간격을 유연하게 처리
