color : 글자 색을 지정하는 속성

        선택자 {
            color : 색상명(영문) | 16진수 숫자 RGB(#fff, #ffffff)
                | rgb(255, 255, 255) | rgba(255, 255, 255, 1)
                | hsl(360, 100, 100) | hsla(360, 100, 100, 1)
            }


        * rgb(빨강 (Red), 초록(Green), 파랑(Blue))
        * hsl(색상(Hue), 채도(Saturation), 명도(Lightness))
        * rgba/hsla에서 a(Alpha)는 투명도 (0 : 투명 , 1 : 불투명)
        * 6개의 색상 지정 방법은 색과 관련된 모든 css 속성에서 사용 가능




  • 색상명으로 지정
  • 16진수 rgb로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정
