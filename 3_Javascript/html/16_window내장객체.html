window 객체

        - 브라우저 창 자체를 나타내는 객체


        - window 객체는 자바스크립트의 최상위 객체이며
        DOM, BOM 으로 분류된다.


        DOM(Document Object Model) : document
        BOM(Browser Object Model) : location, history, screen, navigator

        location => 브라우저의 url 내용을 가져오거나 변경하기 위함
        navigator => 운영체제, 언어, 네트워크 상태

        * window 객체는 창 자체를 나타내고 있으므로
        어디서든 접근이 가능하다.
        그래서 window 라는 객체 호출부를 생략할 수 있다.
        ex) window.alert() == alert()

시간을 지연시키는 방법

Window.setTimeout(함수, 몇 ms지연할래)

지연시간 후에야 함수가 1회 샐행

Window.setInterval(함수, 몇ms 지연할래)

차이가 있다면 지연시간 마다 계속 함수를 실행한다

12: 41: 30

Window.clearInterval(setInterval이저장된 변수)

위 버튼을 누르시면 시계가 멈춰요
