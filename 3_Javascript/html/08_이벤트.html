이벤트(Event)

        이벤트 : 동작, 행위
        -> 브라우저에서 할 수 있는 동작


        ex) click, keyup, keydown, keypress, mouseover, mouseon, mouseout,
            mouseleave, drag, change, submit...


        * 이벤트 리스너(Event Listener)
        -> 브라우저에서 이벤트가 감지 될 때까지 대기하다가
        감지가 되었을 때 연결된 기능(함수)을 수행하는 것


        - 이벤트 리스너는 "on이벤트명" 형식으로 작성됨


        ex) onclick, onchange, onsubmit, onkeyup ...






        * 이벤트 핸들러(Event Handler)
        -> 이벤트 리스너에 연결된 기능(함수)으로
        이벤트가 감지 되었을 때 수행할 내용을 작성한 함수(function)


이벤트 모델의 종류

인라인 이벤트 모델

      - HTML요소에 이벤트리스너 = '이벤트핸들러'
      의 형식으로 작성하는 모든 방식이 인라인이벤트 모델의


      -하나의 요소에 동일한 이벤트 핸들러를 중복으로 작성 불가


      - this => 이벤트가 발생한 현재 요소


      - 사용빈도 낮음

      button onclick="readValue()">입력 /button














고전 이벤트 모델

        -HTML요소가 아닌 JS파일 자체에서 DOM객체에 직접 이벤트 관련 코드를 작성

        - 한 요소에 동일한 이벤트 핸들러 중복 작성X
